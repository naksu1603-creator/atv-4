# Documentação de Acessibilidade – WCAG 2.1 AA

Este projeto foi desenvolvido seguindo as diretrizes WCAG 2.1 nível AA, incluindo:

- Navegação por teclado completa
- Estrutura semântica
- Alto contraste
- Modo escuro acessível
- Foco visível reforçado
- ARIA labels adequados
- Contraste acima de 4.5:1
